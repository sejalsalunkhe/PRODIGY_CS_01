Task-01: Implement Caesar Cipher

 Description
This project is part of my Cyber Security Internship (Track Code: CS).
It implements the Caesar Cipher algorithm in Python to encrypt and decrypt text messages using a user-defined shift value.

 Features
- Encrypt text using custom shift value
- Decrypt encrypted messages
- Preserves uppercase and lowercase letters
- Handles spaces and special characters

Technologies Used
- Python 3
- Object-Oriented Programming

 How to Run
bash
python ciphervault.py
